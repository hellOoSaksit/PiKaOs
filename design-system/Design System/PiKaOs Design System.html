Design System

The PiKaOs design language — calm neutral surfaces, one confident indigo accent, and springy "physical-button" motion. Two themes only: pro (light) and pro-dark (night).

🧊

Calm surfaces

Cool neutral greys carry the UI. Color is reserved, never decorative — surfaces stay quiet so content leads.

🔵

One indigo accent

A single confident indigo (--gold family, named for history) marks every primary action and active state.

🟦

Springy physics

Buttons are 3D "physical" objects — a base shadow, sink on press, overshoot on release via shared spring easings.

🅿️

3D Mitr letters

The brand signature: chunky cartoon glyphs with a stroke-behind-fill and stacked hard shadows.
